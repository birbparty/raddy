Per the Unscii README, the project is Public Domain (CC-0) EXCEPT for the files
derived from or containing parts of Roman Czyborra's GNU Unifont project
(unifont.hex, hex2bdf.pl, and the `unscii-16-full.*` variants), which are GPL.

This asset is `unscii-16.ttf` — the NON-"full" variant — which is in the
Public Domain / CC-0 portion and is freely redistributable. The GPL-licensed
`unscii-16-full.*` and unifont-derived files are deliberately NOT vendored here.

Why this font
-------------
Chosen as raddy's bundled acceptance-test font (bead raddy-8an.3): public-domain,
freely redistributable, and a clean monospaced cell suitable for measuring text
width vs raylib MeasureTextEx in the acceptance spec (raddy-8an.9). It does NOT
reference topdown's assets/fonts/match7.ttf.

Visual legibility sign-off
--------------------------
Per raddy-8an.3, legibility at non-native sizes (8/10/16/20/32 px) is a visual
judgment that requires a real GL/window context — the harness for which is the
acceptance spec (raddy-8an.9), which DEPENDS on this bead. That on-device/human
visual sign-off is therefore deferred to raddy-8an.9's acceptance gate; this
asset provides a valid, redistributable, structurally-correct TrueType file for
that spec to load at two distinct ppem.
